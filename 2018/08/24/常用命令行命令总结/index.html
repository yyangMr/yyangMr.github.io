常用命令行命令总结

发表于 2018-08-24

什么是命令行

我们平时使用计算机的时候，任意打开一个文件夹窗口，即可在计算机硬盘上来回切换，创建文件，目录，删除文件，或是在我们需要的文件之间导航。而用命令行我们可以做同样的事。

命令行界面（英语：command-line interface，缩写：CLI）是在图形用户界面得到普及之前使用最为广泛的用户界面，它通常不支持鼠标，用户通过键盘输入指令，计算机接收到指令后，予以执行。也有人称之为字符用户界面（character user interface, CUI）。

常见的命令行程序

  • bash(Unix 系统)
  • cmd.exe(Windows 系统)
  • Windows PowerShell(支持 .NET Framework 技术的 Windows NT 系统)

正式开始

基本命令

列举文件 ls

ls 是 list 的缩写，使用这个命令可以列举当前文件夹下可见的所有文件和目录。

添加参数 -a 显示所有文件夹。

改变目录 cd

cd 是 changing directories 的缩写，用 cd 命令可以打开任意一个目录。

可以使用 .. 来返回上一级目录。

新建文件 touch
移动文件 mv
文件复制 cp
删除文件 rm
命令行自动补全

在输入命令后，按 Tab 即可自动补全文件名或是目录名。

常用命令

什么是 sudo

sudo 是 superuser do 的缩写，即获取超级用户权限，当命令行命令会影响计算机时(例如删 除全部文件，或者安装程序)，系统会要求在命令前加上 sudo。

新建文件夹 mkdir
移动文件夹 mv

使用 . 将文件夹移动到当前文件夹。

删除文件夹 rm -f

rm 命令无法直接删除一个文件夹，

想要删除文件夹，必须加上参数 -r，-r 是 recursicely 的缩写，这样命令就不仅会删除该文件夹，还会删除该文件夹下的所有文件和文件夹。

删库跑路命令 -rf

f 是 force 的缩写，在这里是强制删除硬盘根目录的意思

当前文件夹 .

将文件移动到当前文件夹。

使用模糊匹配 *

将所有 txt 文件移动到指定文件夹下
