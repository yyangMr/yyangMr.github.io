使用动态规划解决最大子数组问题

发表于 2018-09-04

问题描述：

Given an integer array nums, find the contiguous subarray (containing at least one number) which has the largest sum and return its sum.

Example：

Input: [-2,1,-3,4,-1,2,1,-5,4],
Output: 6
Explanation: [4,-1,2,1] has the largest sum = 6.

Follow up:

If you have figured out the O(n) solution, try coding another solution using the divide and conquer approach, which is more subtle.

问题分析：

很明显，这个问题是一个优化问题。可以通过 DP 解决。使用动态规划，首先要做的就是找到子问题的形式，这对之后找到递归关系式会有帮助。

子问题的形式似乎是：maxSubArray(int A[], int i, int j) 这代表数组 A[i,j] 的最大子数组，这样一来，我们的问题就变成解决 maxSubArray(A, 0, A.length - 1)，然而，如果我们将子问题定义为这种形式，将会很难发现子问题和原问题之间的联系，我们无法将原问题划分为子问题并且用解决子问题的方法去解决原问题。

所以，子问题的形式应该是：maxSubArray(int A[], int i)，即数组 A[0:i] 的最大子数组必须包含元素 A[i] 在末尾。这种形式不如上一种灵活，因为我们加上了限制，元素 A[i] 必须包含在序列内，这样一来，原问题和子问题的关系就很清晰了。

maxSubArray(A, i) =A[i]+ maxSubArray(A, i - 1) > 0 ? maxSubArray(A, i - 1) : 0;
